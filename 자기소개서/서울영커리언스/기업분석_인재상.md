# 서울영커리언스 지원 기업 분석 (인재상·직무·어필 포인트)

> 지원 구성: 1순위 SH · 2순위 더블에이스 · 3순위 여성가족재단
> ⚠️ 공식 인재상 세부 문구는 홈페이지가 이미지로만 게시한 경우가 많아,
>    지원/면접 직전 각 사이트에서 한 번 더 확인 권장 (아래 "추정" 표시 참고)

## 1. SH 서울주택도시개발공사  (1순위 · IT/개발/데이터)

**개요**
- 서울시 100% 출자 지방공기업(공공디벨로퍼). 천만 시민 주거안정·주거복지 담당.
- 주요 사업: 공공주택 공급(공공분양·장기전세·행복주택), 매입임대, 도시재생, 택지개발.
- 디지털 사업: SH 버추얼 메타버스(디지털트윈), 공공데이터 개방, 스마트시티·스마트홈.
- ※ 2025.7.14자 사명 변경: "서울주택도시개발공사" (영문 SH 동일)

**미션 / 비전**
- 미션: "천만 시민과 함께 주거안정과 주거복지에 기여"
- 비전: "집 걱정 없는 고품격 도시 건설을 선도하는 SH"
- ESG: "품격있는 도시조성에 ESG 가치를 더하는 SH"
- 핵심가치/인재상(추정): 공공성·시민(주거복지), 전문성, 혁신, 소통·신뢰, 사회적 가치
  → 정확한 단어는 i-sh.co.kr 「SH소개 > 전략체계」에서 확인

**IT/개발/데이터 직무 (사무직 내 '전산')**
- 정보시스템(청약시스템·임대관리·회계 등) 운영·유지보수·구축
- SH 인터넷청약시스템 등 대시민 온라인 서비스 운영·고도화
- 공공데이터 개방·관리(Open API), 데이터 품질관리
- 정보보안, 전산 인프라(서버·네트워크·클라우드)
- 확장: 디지털트윈/메타버스, 스마트홈(IoT), 프롭테크·주거복지 데이터 분석

**자소서/면접 어필 포인트**
1. 공공성·주거복지 가치 공감 ("기술로 시민의 주거안정에 기여")
2. 데이터 기반 문제해결 (HIRA 공공데이터 경험과 직결!)
3. 디지털전환·스마트시티/프롭테크 이해
4. 시스템 안정성·정보보안·책임감 (대시민 서비스는 무중단·보안 핵심)
5. 소통·협업·사회적 가치(ESG)

**→ 내 강점 매칭: HIRA 공공 의료데이터 분석 = 공공성+데이터 직결**

## 2. 더블에이스 / 로켓펀치(RocketPunch)  (2순위 · IT/개발/데이터)

**개요**
- 커리어·비즈니스 네트워킹 플랫폼 '로켓펀치' 운영. 본사 성수동.
- "사람·기업·기회를 연결"하는 채용·프로필·네트워킹 서비스.
- 누적 개인 프로필 약 45만, 기업 프로필 15만. 글로벌 확장(다국어 자동번역) 추진.

**인재상 / 문화**
- 공식 명문화된 인재상 선언문은 확인 안 됨(추정).
- 서비스 철학 키워드: "전문성을 매개로 연결되어 성장", "언어의 경계 없이 모두와 연결"
- 성수동 스타트업·소규모 IT 기업 → 수평적·자율적·다직무 협업 문화(추정)

**IT/개발/데이터 직무 (추정 — 공식 직무기술서 확인 필요)**
- 채용·프로필·네트워킹 웹/앱 개발·운영, 사용자/기업 데이터 처리·분석
- 검색·매칭·추천 기능 개선, 다국어·결제 연동
- 스택(추정): React/TS(프론트), Python·Node(백), SQL/RDB, AWS

**자소서/면접 어필 포인트**
1. 연결·네트워킹 가치 공감 (로켓펀치 미션과 내 경험 연결)
2. 직접 만들어 본 프로젝트 (결과물 중심, 기술·기여 구체)
3. 데이터로 문제를 해결한 경험
4. 빠른 학습력·주도성 (스타트업·소규모 팀 적합성)
5. 글로벌·확장 마인드

**→ 내 강점 매칭: BAMBOO(동아리 '연결' 플랫폼) 직접 개발 = 미션과 딱**

## 3. 서울시 여성가족재단  (3순위 · 엔지니어링/연구개발)

**개요**
- 서울특별시 출연기관. 여성·가족 정책 연구 + 공공사업 운영.
- 4대 사업: 저출생대응(일생활균형·여성일자리), 돌봄, 양성평등(폭력예방·디지털성범죄 지원), 정책연구(성인지 통계시스템).

**미션 / 비전 / 핵심가치 (공식)**
- 미션: "양성평등 행복도시 서울 실현"
- 비전: "여성·가족을 위한 일·돌봄·안전 지원 플랫폼"
- 핵심가치: **양성평등 · 신뢰 · 창의 · 안전**
- ⚠️ "조직문화 개발자"는 이 재단 인재상 아님(혼동 주의). 위 4가지가 공식.

**'엔지니어링/연구개발' 직무 (= 데이터/정책연구, 추정)**
- 성인지 통계·성별분리통계 수집·가공·시각화
- 여성·가족·돌봄·일자리 실태조사 설문 설계, 데이터 정제·분석
- 정책연구보고서·리뷰 리서치 보조(통계·인포그래픽)
- 요구 역량: Excel·SPSS·R·Python, 사회조사방법론·기초통계, 보고서 작성

**자소서/면접 어필 포인트**
1. 공공성·공익 마인드 (데이터로 시민 삶 개선)
2. 성평등·돌봄 가치 공감 (핵심가치 '양성평등')
3. 데이터 기반 정책연구 역량 (근거 기반 evidence-based)
4. 창의·혁신 (데이터 시각화·새 분석 방법)
5. 신뢰·협업·꼼꼼함 (공공기관 정확성·책임감)

**→ 내 강점 매칭: HIRA 데이터 분석·예측 모델링 = 데이터 기반 정책연구 직결**
