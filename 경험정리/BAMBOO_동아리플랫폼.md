# BAMBOO — AI·데이터 동아리 내부 활동 플랫폼

> 동아리원들이 활동을 공유하는 웹 플랫폼 (React · TypeScript)
> **내 역할: 프론트엔드 개발** (깃허브 커밋으로 검증 가능, heejungJang04)
> ※ 현재 서버는 내려가 있음 → 코드는 깃허브에 남아있음

## 한 일 (깃허브 커밋 기준, 2025.7~11)
- **중복 제출 방지** (studyedit, 커리큘럼/스터디 add·edit·delete)
- **댓글 수 실시간 반영**
- **기본 프로필 설정** 기능 추가 (마이페이지)
- **입력 글자 수 제한** (게시 글자수 제한)
- UI 스타일링 (editor.css, post.module.css, 폰트)

## 강조 포인트
- 단순히 화면만 그린 게 아니라 **잘못된 입력·중복 데이터를 막아 UX와 데이터 품질까지 고려**

## 자소서/면접 활용 포인트
- **개발 직무**: "직접 만든 서비스" 대표 사례
- ⚠️ 면접 대비: '중복 제출 방지'를 코드로 어떻게 구현했는지 한 문장으로 정해둘 것
  (예: 제출 버튼 비활성화 / 로딩 상태 관리 / 중복 요청 차단 등)
- "서버는 내렸지만 코드는 깃허브에 있다"로 검증 가능
